# Menu Management - Quick Start Guide for Content Managers

## üéØ What is the Menu System?

The Menu System allows you to create and manage all navigation menus on your website without touching any code. You can:
- Create header menus
- Build footer link columns
- Design mobile navigation
- Add dropdown menus
- Create mega menus with special layouts

## üìç Accessing the Menu Manager

1. Log into the Admin Dashboard
2. Click **"Menus"** in the sidebar under "Content"
3. You'll see a list of all your menus

## ‚ú® Creating Your First Menu

### Step 1: Click "Create Menu"
Click the blue "+ Create Menu" button in the top right.

### Step 2: Fill in Basic Information
- **Menu Name**: Give it a descriptive name (e.g., "Main Navigation")
- **Location**: Choose where this menu will appear:
  - **Header - Main Navigation**: Your main site menu
  - **Footer - Column 1-4**: Footer link columns
  - **Mobile - Main Menu**: Mobile drawer menu
  - **Mobile - Bottom Navigation**: App-style bottom bar

### Step 3: Add Menu Items
Click the "+ Add Item" button to add your first link.

For each item, you can set:
- **Label**: The text that appears (e.g., "Home", "About Us")
- **Type**: Choose from:
  - **Link**: Standard navigation link
  - **Dropdown**: Opens a submenu
  - **Mega Menu**: Large dropdown with special layouts
  - **Button**: Styled as a button (for CTAs)
- **URL**: Where the link goes (e.g., `/about`, `/contact`)
- **New Tab**: Check this to open in a new window

### Step 4: Reorder Items
Simply **drag and drop** items using the ‚ò∞ handle to reorder them.

### Step 5: Preview Your Menu
Look at the **Preview** panel on the right to see how your menu will look.

### Step 6: Publish
1. Make sure "Active" is checked
2. Click "Create Menu"
3. Your menu is now live!

## üé® Menu Types Explained

### 1. Standard Link
The most common type. Just a simple link.

**Example:**
- Label: "About Us"
- URL: `/about`
- Type: Link

### 2. Dropdown Menu
Creates a menu with sub-items.

**How to create:**
1. Set Type to "Dropdown"
2. Click "+ Add Child Item"
3. Add as many child items as you need
4. Each child is a regular link

**Example:**
- Label: "Services"
- Type: Dropdown
- Children:
  - "Gratuity Calculator" ‚Üí `/`
  - "MOHRE Calculator" ‚Üí `/mohre-calculator`
  - "DIFC Calculator" ‚Üí `/dubai/free-zones/difc`

### 3. Mega Menu
A large dropdown with special layouts.

**Available Layouts:**
- **Emirates Grid**: Shows UAE emirates with popular areas
- **Calculators List**: Displays all calculator types with icons
- **Services Columns**: Multi-column service listing

**How to create:**
1. Set Type to "Mega Menu"
2. Choose a layout from the dropdown
3. The content is automatically generated

**Example:**
- Label: "Locations"
- Type: Mega Menu
- Layout: Emirates Grid

### 4. Button
Styled as a call-to-action button.

**Example:**
- Label: "Calculate Now"
- Type: Button
- URL: `/#calculator`

## üéØ Best Practices

### Header Menus
- Keep it simple (5-7 items max)
- Use dropdowns for related content
- Put important actions as buttons
- Order: Home ‚Üí Services ‚Üí Resources ‚Üí About ‚Üí Contact

### Footer Menus
- Organize by category (Company, Services, Legal, Support)
- Use 3-4 columns
- Include important links like Privacy Policy
- Keep each column to 5-6 links

### Mobile Menus
- Simplify your header menu
- Remove nested dropdowns if possible
- Use clear, short labels
- Consider a bottom nav for key actions

## üí° Pro Tips

### Adding Badges
You can add small badges to highlight items:
1. Edit a menu item
2. Add badge text (e.g., "New", "Beta")
3. Choose a color (primary, success, danger, warning)

**Example:**
- Label: "New Feature"
- Badge: "Beta"
- Badge Color: Warning (yellow)

### Opening in New Tab
Check the "New Tab" option for:
- External links
- PDF downloads
- Social media links
- Partner websites

### Mega Menu Tips
- Use "Emirates Grid" for location-based navigation
- Use "Calculators List" to showcase all your tools
- Use "Services Columns" for general service categories

Your changes appear immediately on the website!

## üîç Troubleshooting

### My menu isn't showing on the website
- Check that the menu is marked as "Active"
- Verify you selected the correct location
- Make sure you have at least one item in the menu

### Dropdown isn't working
- Ensure the item type is set to "Dropdown"
- Add at least one child item
- Check that you've added child items correctly

### Preview looks different from website
- The preview is a simplified version
- Actual styling may vary based on your theme
- Test on the live site to see final result
